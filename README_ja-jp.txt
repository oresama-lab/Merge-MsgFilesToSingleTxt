# Merge-MsgFilesToSingleTxt

このPowerShellスクリプトは、GUIを使用してユーザーが指定したフォルダ内のすべての.msgファイルを一つの.txtファイルに変換します。ユーザーはフォルダ選択ダイアログとファイル選択ダイアログを通じて入力フォルダと出力テキストファイルの場所を指定することができます。変換が完了すると、ポップアップメッセージが表示されます。

## 使い方

1. スクリプトをPowerShellで実行します。
2. 「Select Folder」ボタンをクリックして、.msg ファイルが保存されているフォルダを選択します。
3. 「Select Output File」ボタンをクリックして、変換された内容を保存する.txt ファイルの場所と名前を指定します。
4. 「Convert」ボタンをクリックして変換を開始します。
5. 変換が完了すると、ポップアップメッセージが表示されます。
